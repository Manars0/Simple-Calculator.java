# Simple Calculator (Java Swing GUI)

A simple Java GUI calculator built using the **Swing** library.  
The program allows the user to enter three numbers and perform basic arithmetic operations: **addition, subtraction, multiplication, and division.**

This project demonstrates the use of basic GUI components, event handling, and input validation.

## Features
- Simple and user-friendly interface  
- Performs four operations: ADD, SUB, MUL, DIV  
- Handles division by zero errors  
- Clean design with organized layout using `JPanel` and `GridLayout`  
- Lightweight and easy to run  

## GUI Overview
The interface contains:
- Three input fields for numbers  
- Four buttons for operations (ADD, SUB, MUL, DIV)  
- A result label to display the output  

## Requirements
- Java (JDK 17 or later)  
- Any IDE that supports Java (e.g., Eclipse, IntelliJ, VS Code)
